Numpy:
    - library for the Python programming language
    - Adding support for large, multi-dimensional arrays and matrices
    - Along with a large collection of high-level mathematical functions to operate on these arrays

Benefits of Numpy:
    - Uses much less memory to store data
    - Creating n-dimension arrays
    - Supports mathematical operation on n-dimesnion array
    - Finding Elements in NumPy array

Most Usefull methods in Numpy:
    - array()
    - arange()
    - ones()
    - pad()
    - asarray()
    - reshape()
    - unique()
    - linspace()
    - column_stack()
    - row_stack()
    - sum()
    - nditer()
    - empty()
    - append()
    - vstack()

Most Usefull Variables in Numpy:
    - size
    - itemsize
    - nbytes
    - dtype
